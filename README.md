## 1. 프로젝트 소개

Fire&Water 게임을 레퍼런스하여 만든 2D 플랫포머 게임입니다.
다음과 같은 기능을 구현하였습니다.

 - 캐릭터 이동 구현 및 상호작용
 - 협력 기반 퍼즐 요소
 - TileMap 맵 설계 및 UI 구성
 - 점수, 맵 선택 등 다양한 게임 시스템
